## Install

```html
<script src="path/to/sync-console.js"></script>
<script>
var syncConsoleManager = new SyncConsoleManager({
    project: 'im',
    maxLogCount: 50,
    server: 'http://sync.bood.in/',
})
</script>
```


## url query options

- _sync_console_show 
- _sync_console_token
- _sync_console_project
